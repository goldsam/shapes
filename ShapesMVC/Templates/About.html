About

This application was written by Samuel Goldmann on behalf of the wonderful people at eDocument Solutions.

Backend

This application's backend was implemented with ASP.NET Web API 2 and uses the ASP.NET Entity Framework for its object-relational mapping.

Backend development time was approximately 10 hours, which included:

  • 3 hours to reseaerch and learn the Entity Framework and related best practices
  • 7 hours to implement and rigorously test/debug shape generation components, data models, and route controllers.

Frontend

The application's frontend was built using AngularJS and Bootstrap. Additionally, AngularJS UI - Bootstrap components were used to provide a simple pagination widget over the shape history.

Frontend development time took approximately 8 hours, which included:

  • 5 hours hours to design and implement the UI, and integrate 3rd party libraries.
  • 3 hours to implement and rigorously test/debug the view model and route controllers.
